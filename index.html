仮想APIリファレンス

🚀 基本イベント

onStart(() => { ... })

ゲーム起動時に1回だけ実行される初期化用イベントだよ。

onTick(() => { ... })

1秒間に約60回、画面更新ごとにループ実行されるメイン処理だよ。

👑 メインAPI (api.*)

api.makeObject(options)

安全な初期値を持ったゲームオブジェクト（pos, rotate, size, hitBox）を生成するよ。

api.isHit(obj1, obj2)

2つのオブジェクトの衝突を判定するよ。矩形の回転（OBB）にもバッチリ対応！

api.stamp(costumeName, object)

インポートした画像を使って、指定オブジェクトをカメラ空間にスタンプ描画するよ。

api.playSound(soundName)

インポートしたサウンド音源を、最初から再生するよ。

api.setCamera(x, y, z, rotate, size)

カメラの位置や倍率を安全に一括変更するよ。

🎨 描画シリーズ (api.display.*)

.rectangle(pos, size, rotate, style, isCam)

回転可能な矩形（四角形）を描画するよ。styleで色なども指定可能。

.circle(pos, radius, style, isCam)

指定した位置に真円を描画するよ。

.triangle(p1, p2, p3, style, isCam)

3点をもとに三角形を描画するよ（Zクリップ安全ガード付き）。

.line(p1, p2, style, isCam)

指定した2点間に直線を引くよ。

.text(pos, text, fontSize, rotate, style, isCam)

指定位置に回転可能な文字を表示するよ。
